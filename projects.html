Remote Control Submarine

  • Designed and built dual ballast system using DC motors with encoders to drive worm gears and racks, changing plunger position in syringe connected to external water, allowing the submarine's density to change on command
  • Designed and built servo gear mechanism to swivel rear thruster, keeping the servo dry within the submarine by using a magnetic coupling to transmit the force.
  • 3D printed frame and assembled everything
  • Soldered custom breakout board for Linux development board
  • Due to poor PWM output from Linux development board, used Arduino Uno to control PWM outputs, and established UART connection between Aduino and Linux development board to continually send data
  • Harnessed thrusters, ballast motors, ballast encoders, ballast limit switches, H-bridge circuit, buck converter, Arduino, and Linux development board
  • Harnessed submarine with 12V cable for power, and ethernet cable for data streaming between laptop and Linux development board
  • Developed custom control software in Python for Linux board to receive controller input from laptop, and update position of motors on the submarine, allowing for easy control of yaw, pitch, roll, and thrust
  • Used C++ script on Arduino to arm all ESCs and continually update PWM output based on serial data received from Linux board

3D Printed Brushless DC Motor and Driver
